# FieldArrayStore

Type that defines the state of a form field array.

## Definition

- `FieldArrayStore` <Property type="object" />
  - `name` <Property type="string" />
  - `items` <Property {...properties[isPreact() ? 'preact' : 'default'].items} />
  - `error` <Property {...properties[isPreact() ? 'preact' : 'default'].error} />
  - `active` <Property {...properties[isPreact() ? 'preact' : 'default'].active} />
  - `touched` <Property {...properties[isPreact() ? 'preact' : 'default'].touched} />
  - `dirty` <Property {...properties[isPreact() ? 'preact' : 'default'].dirty} />

### Explanation

The `items` do not contain the value of the fields, but only a random unique number.<Qwik> You should pass this value in the loop to the first JSX element as `key` property.</Qwik>
