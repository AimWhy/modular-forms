# Installation

You can add Modular Forms to your project via `npm`.

<Solid block>

> If you use `yarn` or `pnpm`, just replace the installation command at the beginning.

```bash
npm install @modular-forms/solid
```

</Solid>

<Qwik block>

> The package is currently not yet published. We are currently working on the final details. However, the code is already on <a href={`${import.meta.env.VITE_GITHUB_URL}/tree/main/packages/qwik`} target="\_blank" rel="noreferrer">GitHub</a>.

```bash
npm install @modular-forms/qwik
```

</Qwik>

After that you can import it into any JavaScript or TypeScript file.

<Solid block>

```ts
import { … } from '@modular-forms/solid';
```

</Solid>

<Qwik block>

```ts
import { … } from '@modular-forms/qwik';
```

</Qwik>
