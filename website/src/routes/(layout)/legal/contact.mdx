# Contact

Modular Forms is an open source project led by Fabian Hiller and developed publicly on GitHub by more than 20 contributors. The project is non-commercial and the source code is available under the MIT licence.

## General inquiries

For general inquiries, such as questions, bug reports and feature requests, please [create an issue](https://github.com/fabian-hiller/modular-forms/issues/new) on GitHub.

## Security issues

To report a vulnerability, please contact us via hello@fabianhiller.com. We recommend that you use the latest versions of the library to ensure that your application remains as secure as possible.

## Partner requests

If you are interested in a partnership with Modular Forms, please contact us at hello@fabianhiller.com. We are open to collaborations that help us improve the project and make it more accessible to a wider audience.
