# remove

Removes a item of the field array.

```ts
remove(form, name, options);
```

## Parameters

- `form` <Property {...properties[getFramework()]?.form} />
- `name` <Property type="string" />
- `options` <Property type="object" />
  - `at` <Property type="number" />

### Explanation

With `at` you define the index of the item to be removed. The action is executed only if you specify a valid index.
