# FieldState

Type that defines the state of a form field.

## Definition

- `FieldState` <Property type="object" />
  - `props` <Property type="object" />
    - `name` <Property type="string" />
    - `ref` <Property {...properties.ref} />
    - `onInput` <Property {...properties.onInput} />
    - `onChange` <Property {...properties.onChange} />
    - `onBlur` <Property {...properties.onBlur} />
  - `name` <Property type="string" />
  - `value` <Property {...properties.value} />
  - `error` <Property type="string" />
  - `active` <Property type="boolean" />
  - `touched` <Property type="boolean" />
  - `dirty` <Property type="boolean" />

### Explanation

The `props` should be spread into a HTML input, select or textarea element as shown in the following example.

```tsx
<input {...field.props} type="text" />
```

The state properties of the field are reactive and can be used, for example, in the JSX area or Solid's `createEffect` primitive.
