# setResponse

Sets the response of the form.

```ts
setResponse(form, response, options);
```

## Parameters

- `form` <Property {...properties.form} />
- `response` <Property {...properties.response} />
  - `status` <Property {...properties.status} />
  - `message` <Property type="string" />
- `options` <Property {...properties.options} />
  - `duration` <Property {...properties.duration} />

### Explanation

To reset the `response` state of the `form`, simply set the `response` parameter to an empty object.

If you want the `response` to be automatically reset after a certain time, you can add a `duration` in milliseconds to the `options`. If the response changes in the meantime of the specified duration, the reset will be skipped.

export const properties = {
  form: {
    type: {
      type: 'custom',
      name: 'ModularForm',
      href: '/api/ModularForm',
    },
  },
  response: {
    type: ['object', { type: 'object', entries: [] }],
  },
  status: {
    type: [
      { type: 'string', value: 'info' },
      { type: 'string', value: 'error' },
      { type: 'string', value: 'success' },
    ],
  },
  options: {
    type: 'object',
    defaultValue: { type: 'object', entries: [] },
  },
  duration: {
    type: ['number', 'undefined'],
  },
};
